# Changelog

## v1.1.0

### Fixed

- 修复 Windows 非 100% 缩放下的取色坐标偏移。
- 修复多显示器、主屏不在左上角和负坐标布局下的取色错误。
- 修复键盘确认依赖错误窗体坐标的问题。
- 移除可能导致鼠标被异常限制的 `ClipCursor` 逻辑。
- 修复覆盖层事件回调期间提前释放窗体资源的潜在异常。
- 为剪贴板被其他进程暂时占用的场景增加重试与失败提示。

### Improved

- 升级为 Per-Monitor V2 DPI 感知。
- 使用全虚拟桌面覆盖层与屏幕物理坐标采样。
- 增加不抢焦点的取色悬浮预览。
- 重构主界面为现代、紧凑的卡片式布局。
- 增加 HEX / RGB 独立复制按钮与更清晰的状态反馈。
