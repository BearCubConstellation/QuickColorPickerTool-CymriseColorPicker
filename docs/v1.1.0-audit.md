# v1.1.0 验收清单

| 场景 | 操作 | 预期 |
|---|---|---|
| 单屏 100% | 取 10 个不同区域颜色 | 鼠标像素与结果一致 |
| 单屏 125% / 150% / 175% / 200% | 重复取色 | 无固定方向偏移 |
| 双屏同缩放 | 依次取主屏、扩展屏颜色 | 两块屏幕都可取色 |
| 双屏不同缩放 | 例如主屏 150%、副屏 100% | 跨屏移动和取色准确 |
| 负坐标布局 | 将副屏放在主屏左侧或上方 | 左/上副屏可正常取色 |
| 键盘确认 | 移动鼠标后按 Space / Enter | 结果与当前指针位置一致 |
| 取消 | 右键、Esc、Alt+F4 | 恢复主窗口，不残留覆盖层或浮窗 |
| 剪贴板占用 | 连续快速取色与复制 | 有明确成功/失败提示，不崩溃 |
| 窗口关闭 | 取色中关闭主程序 | 鼠标不被锁定，进程正常退出 |
