# Cymrise Color Picker v1.1.0

这是一次以“取色准确性”和“桌面体验”为核心的更新。

## 修复

- 修复 Windows 125%、150% 等缩放场景下，鼠标位置与实际取色位置不一致的问题。
- 修复扩展屏、负坐标布局、跨屏取色失效或偏移的问题。
- 修复键盘确认取色使用错误坐标的问题。
- 移除光标裁剪，避免异常情况下鼠标被限制在某一区域。

## 优化

- 升级为 Per-Monitor V2 DPI 感知。
- 新增取色过程中的悬浮颜色预览。
- 主界面改为简洁现代的卡片式布局。
- 新增 HEX / RGB 独立复制与更明确的状态反馈。
- 新增 GitHub Actions 自动构建流程。
