# ColorPicker v1.3.0

## English

- Added a dedicated English executable: `ColorPicker-en.exe`.
- Compressed the title area and bottom spacing for a smaller utility-style interface.
- The header title starts white and changes to the confirmed picked color.
- Removed duplicate legacy source from the active build path.
- Release now contains separate Chinese and English Windows x64 packages.

## 中文

- 新增独立英文版程序：`ColorPicker-en.exe`。
- 进一步压缩标题区与底部行间距，使界面更符合小工具定位。
- 顶部标题初始为白色，确认取色后同步变为所选颜色。
- 清理活动构建路径中的重复旧源码。
- Release 现在同时提供中文与英文 Windows x64 独立压缩包。
