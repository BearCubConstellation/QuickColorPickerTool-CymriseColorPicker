# Cymrise Color Picker v1.2.2

## 界面修复

- 默认窗口扩大，并改为按 DPI 与客户区尺寸动态布局。
- 修复标题、颜色值、复制按钮、提示语、状态语和底部信息的拥挤、遮挡与裁剪。

## 取色响应优化

- 移除启动取色时同步复制完整虚拟桌面的操作。
- 改为直接读取当前鼠标物理坐标的单像素颜色。
- 多显示器、不同缩放比例和负坐标布局仍使用同一物理坐标系。
- 点击“开始取色”后应立即进入取色状态，不再出现数秒等待。

## 发布

- 原生 Win32 x64 单文件。
- 无 .NET Runtime、PDB、`.config` 或第三方运行时依赖。
