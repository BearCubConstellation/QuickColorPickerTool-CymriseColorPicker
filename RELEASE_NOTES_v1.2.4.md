# ColorPicker v1.2.4

## 紧凑化

- 默认窗口由 540 × 420 DIP 调整为 410 × 320 DIP。
- 缩短标题区、卡片间距、按钮高度与底部间距，更符合轻量小工具定位。

## 取色冻结修复

- 点击“开始取色”后会截取当前完整虚拟桌面，并以不透明覆盖层显示该快照。
- 覆盖层会捕获鼠标，桌面及其他窗口不能再接收点击。
- 颜色读取改为从冻结快照采样，取色结果与看到的画面一致。
- 当前颜色浮窗增加鼠标捕获和 16ms 定时刷新，能够持续跟随鼠标。

## 其他

- GitHub 项目入口更新为当前 `BearCubConstellation/ColorPicker` 仓库。
- Windows x64 原生单文件发布，不包含 .NET Runtime、PDB、`.config` 或外部运行时依赖。
